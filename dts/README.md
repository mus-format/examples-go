# DTS Examples

This directory demonstrates features of the [dts-go](https://github.com/mus-format/dts-go)
library.

- [different](different/): Shows how to use a single endpoint to handle 
  different data types.
- [oneof](oneof/): Shows how to serialize and deserialize Go interfaces.
- [versioning](versioning/): Shows how to use a single endpoint to handle 
  different data versions.
